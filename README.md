## Pre-requisites
- Node.js v20.^

## Setting up
```sh
 npm i && npm run setup
```

## Script commands
```sh
# install necessary dependencies
npm run setup

# start app
npm run start

# lint app
npm run lint

# test app
npm run test

# test production
npm run post-production
```
